LES RÈGLES DU JEU

1. Le bouton PLAY fait tourner les roues. Les combinaisons sur lignes de paiement débouchent sur des gains, conformément au tableau des gains.
2. Le bouton $ permet d'afficher le tableau des gains.
3. Le bouton STOP fait arrêter les roues.
4. Lorsque le bouton AUTO est activé, vous jouez automatiquement avec la même mise que celle placée avant le bouton PLAY. Appuyez sur ce même bouton pour désactiver le mode auto.

LIGNES DE PAIEMENT

Tous les symboles sont comptabilisés de gauche à droite, en partant de la roue gauche. Tous les gains sont multipliés par la mise en ligne. Les erreurs rendent tous les gains invalides.
